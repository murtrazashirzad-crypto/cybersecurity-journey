# Contributing

Thanks for your interest in this repo! This is primarily a **personal learning journal** as I work toward OSCP, but contributions are welcome.

## What I’m Happy to Receive

- **Corrections** — if I’ve explained something inaccurately, please open an issue or PR
- **Better techniques** — if there’s a cleaner / more realistic way to solve a room
- **Typos & formatting fixes**
- **Additional references** — links to blog posts, CVEs, or tools that add context

## What I’d Rather You Didn’t Do

- **Don’t post flag values.** TryHackMe rooms are learning tools. Redact flags in any PR.
- **Don’t submit writeups for rooms I haven’t done yet** — I want to do them myself first.

## How to Contribute

1. Fork the repo
1. Create a branch: `git checkout -b fix/typo-in-blue-writeup`
1. Make your changes — follow the format in [`templates/writeup-template.md`](./templates/writeup-template.md)
1. Commit with a clear message: `git commit -m "Fix: clarify EternalBlue payload step"`
1. Push and open a Pull Request

## Found Something Off?

Open an [issue](https://github.com/murtrazashirzad-crypto/-tryhackme-writeups-/issues) — I’d rather know than not.
